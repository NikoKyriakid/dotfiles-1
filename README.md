# dotfiles

* .vimrc
* .zshrc
* .tmux.conf
* .gitconfig
* .gitignore

.vimrc and .zshrc requires some sources in \*.d directory

## install

### how to install

    $ git clone https://github.com/rinx/dotfiles.git ~/.dotfiles

### setup.sh

this script makes symbolic link in home

    $ sh setup.sh
